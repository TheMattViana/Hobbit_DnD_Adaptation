# Asset Mapping for Hobbit Adventures Webapp

## Skill-Related Assets
- **mushrooms.png** → "Mushroom Finding & Foraging" skill
- **wizard_staff.png** → "Granny Weatherwax's 'Persuader'" weapon and magic-related skills
- **sting_sword.png** → "Tookish Courage (Short Sword)" and combat skills
- **forest_tree.png** → Nature and outdoor skills, Wanderer calling background
- **mountain.png** → Exploration and travel-related skills
- **hobbit_hole.png** → Header background (already implemented)
- **wooden_frame.png** → UI frame for skill displays and character information
- **one_ring.png** → Special items and rare treasures
- **lembas_bread.png** → "Waybread Baker" calling and food-related skills
- **ale_mug.png** → Social skills and Hearth-friend calling

## Character-Related Assets
- **hobbit_idle.png** → Default character portrait
- **hobbit_blonde.png** → Alternative character portrait
- **gandalf.png** → NPC or ally display
- **gollum.png** → Enemy display in combat section
- **elf_archer.png** → Ally or NPC for ranged combat
- **dwarf_warrior.png** → Ally or NPC for melee combat

## Animation Assets
- **hobbit_dance_frame1.png** through **hobbit_dance_frame4.png** → Animation for celebration or success
- **fireworks_frame1.png** and **fireworks_frame2.png** → Animation for critical success or special achievements

## Ambiance Assets
- **campfire.png** → Rest and recovery features, footer element
